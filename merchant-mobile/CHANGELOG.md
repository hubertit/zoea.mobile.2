# Merchant Mobile App Changelog

All notable changes to the Zoea Merchant Mobile App will be documented in this file.

## [1.3.0] - 2024-12-20

### Added
- Enhanced booking management features
- Improved analytics dashboard

### Changed
- Updated UI components for better usability
- Performance optimizations

## [1.2.0] - 2024-11-15

### Added
- **Booking Management**:
  - View all bookings
  - Accept/reject bookings
  - Booking notifications
- **Menu Management**: Restaurant menu creation and editing

### Changed
- Enhanced dashboard with key metrics
- Improved navigation

## [1.1.0] - 2024-10-10

### Added
- **Listing Management**:
  - Create and edit listings
  - Upload images
  - Manage amenities
- **Profile Management**: Merchant profile editing

### Changed
- Improved dashboard layout
- Enhanced data synchronization

## [1.0.0] - 2024-09-01

### Added
- Initial release
- **Authentication**: Merchant login and registration
- **Dashboard**: Overview of business metrics
- **Notifications**: Basic notification system
- **Profile**: View merchant profile

### Technical
- Flutter 3.4.3+ setup
- Riverpod for state management
- GoRouter for navigation
- API integration with backend
