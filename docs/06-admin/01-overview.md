# Admin Dashboard - Overview

**Last Updated**: December 30, 2024  
**Technology**: Next.js (React, TypeScript)

---

## Description

The Zoea Admin Dashboard is a Next.js-based web application for platform administrators, partners, and operators to manage the Zoea platform, including users, listings, bookings, analytics, and content moderation.

---

## Technology Stack

- **Framework**: Next.js
- **Language**: TypeScript
- **UI Library**: React
- **Styling**: Tailwind CSS
- **Charts**: ApexCharts
- **Database**: MySQL (legacy)

---

## Key Features

- Dashboard with analytics
- User management
- Listing management
- Booking management
- Content moderation
- Analytics and reporting
- Merchants module
