# Testing Documentation

This directory contains all testing-related documentation for the backend API.

## Files

- **ENDPOINT_TEST_RESULTS.md** - Complete test results for user data collection endpoints
- **TRACKING_ENDPOINTS_TEST_RESULTS.md** - Test results for analytics/tracking endpoints
- **TESTING_GUIDE.md** - Comprehensive guide for testing endpoints
- **ENDPOINT_TESTING_PLAN.md** - Testing checklist and plan

## Quick Start

See `TESTING_GUIDE.md` for instructions on how to test endpoints.

## Test Scripts

Test scripts are located in `/scripts` directory:
- `test-all-endpoints.sh` - Comprehensive endpoint testing script
- `test-endpoints.sh` - Basic endpoint testing script
- `test-all-admin-endpoints.sh` - Admin endpoints testing script
