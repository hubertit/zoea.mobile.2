## [2.0.0] - 2025-01-02

### Mobile App
- **Dark Mode Support**: Complete dark mode implementation
  - Light/Dark/System theme options
  - Theme persistence across sessions
  - Theme-aware logos and components
- **UI Improvements**: Enhanced theming and color consistency

### Backend API
- All endpoints stable and documented
- Swagger documentation maintained

### Admin Dashboard
- Merchant management features
- Analytics dashboard updates
