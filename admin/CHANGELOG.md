# Admin Dashboard Changelog

All notable changes to the Zoea Admin Dashboard will be documented in this file.

## [1.4.0] - 2024-12-20

### Added
- Enhanced merchant management features
- Improved analytics visualizations

### Changed
- Updated dashboard components
- Performance optimizations

## [1.3.0] - 2024-11-25

### Added
- **Analytics Dashboard**:
  - Revenue charts
  - Booking statistics
  - User growth metrics
  - Popular listings analytics
- **Reporting System**: Generate and export reports

### Changed
- Enhanced charts with ApexCharts
- Improved data visualization

## [1.2.0] - 2024-10-20

### Added
- **Merchant Management**:
  - Approve/reject merchant applications
  - View merchant details
  - Manage merchant status
  - Merchant verification
- **Listing Management**:
  - Review and approve listings
  - Feature listings
  - Listing moderation

### Changed
- Improved merchant dashboard
- Enhanced search and filtering

## [1.1.0] - 2024-09-30

### Added
- **User Management**:
  - View all users
  - User details
  - User activity tracking
- **Content Moderation**:
  - Review management
  - Report handling

### Changed
- Enhanced admin interface
- Improved navigation

## [1.0.0] - 2024-09-01

### Added
- Initial release
- **Authentication**: Admin login
- **Dashboard**: Overview of platform metrics
- **Merchants Module**: Basic merchant listing
- **Settings**: Admin settings

### Technical
- Next.js 14 setup
- TypeScript
- Tailwind CSS
- MySQL database connection (legacy)
- API integration with backend

---

## Technical Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Charts**: ApexCharts
- **Database**: MySQL (transitioning to PostgreSQL)
